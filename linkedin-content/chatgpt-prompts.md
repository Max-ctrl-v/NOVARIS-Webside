# ChatGPT / DALL-E Prompts für LinkedIn Post-Bilder

Nutze diese Prompts in ChatGPT (mit DALL-E) oder Midjourney, um passende Bilder zu generieren.
Alle Bilder sollten 1200x1200px (LinkedIn optimal) sein.

## Farbpalette für alle Prompts:
- Background: Dark navy (#0F2747)
- Accent: Blue (#1E6DB5, #4FA3E3)
- Text: White
- Style: Clean, corporate, modern, minimal

---

## PROMPT: Statistik / Zahlen-Posts
```
Create a clean, modern corporate infographic on a dark navy blue background (#0F2747).
Large bold white text showing "[ZAHL]" centered.
Below it smaller text "[BESCHREIBUNG]".
Subtle blue gradient glow (#1E6DB5 to #4FA3E3) behind the number.
Minimal geometric lines and dots as decoration.
No logos. Professional consulting aesthetic. 1200x1200px.
```

Ersetze [ZAHL] und [BESCHREIBUNG] je nach Post:
- "35%" / "F&E-Kosten zurück"
- "4,2 Mio. €" / "Max. Förderung pro Jahr"
- "100%" / "Erfolgsquote"
- "12 Mio. €" / "Bemessungsgrundlage"
- "60%" / "Auftragsforschung förderfähig"

## Video/Animation Prompts (für ChatGPT Video oder Canva)

### Animierte Zahlen
```
Create a short 5-second looping animation:
A large number counting up from 0 to [ZAHL] on a dark navy background.
Blue glow effect (#4FA3E3) pulsing behind the number.
Smooth easing animation. Professional corporate style.
```

### Prozess-Animation
```
Create a 10-second animation showing 3 steps appearing sequentially:
Step 1 slides in from left → Step 2 appears in center → Step 3 from right.
Connected by a growing blue line.
Dark navy background. Blue accents. Corporate style.
```

### Branchen-Slider
```
Create a 15-second animation cycling through 9 industry icons:
Each icon appears with a subtle fade and scale animation.
Dark navy background, blue accent glow.
Icons: gears, code, lightning, car, pill, heart+cross, plane, leaf, building.
Loop seamlessly.
```

## Canva Alternative

Für schnellere Erstellung: Nutze Canva mit diesen Einstellungen:
- Größe: 1200 x 1200 px
- Background: #0F2747
- Akzentfarbe: #4FA3E3
- Font Heading: Montserrat Bold
- Font Body: Inter Regular
- Template-Stil: "Corporate Dark" oder "Tech Startup"

Canva hat auch Video-Templates die du mit den Brand-Farben anpassen kannst.
